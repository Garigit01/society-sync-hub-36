# Society Hub 🏢

Society Hub is a smart infrastructure and management web application designed to streamline residential operations. It features an automated system for tracking maintenance expenses, logging citizen/resident complaints, and managing tasks with a dedicated Super Admin and Resident role split.

## 🚀 Features

* **Dual Dashboard System:** * **Super Admin:** Set monthly maintenance, log payment statuses, track withdrawals, and update complaint progress.
    * **Residents:** Complete mandatory profile setups and view real-time society financial logs.
* **Database Integration:** Powered by Supabase PostgreSQL for real-time live data persistence.
* **Dynamic Actions:** One-click WhatsApp alert routing to seamlessly message residents regarding pending updates.

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, Tailwind CSS
* **Backend & Database:** Supabase (PostgreSQL)
